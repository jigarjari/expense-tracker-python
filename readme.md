# Expense Tracker System

A Python-based Expense Tracker application with SQLite database integration, analytics, charts, and PDF report generation.

## Features

- Add expenses
- Store data using SQLite
- Expense analytics
- Charts and visual reports
- PDF report export

## Technologies Used

- Python
- SQLite
- Pandas
- Matplotlib
- ReportLab
- CustomTkinter

## Project Status

Currently under development.
